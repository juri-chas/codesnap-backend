# Codesnap Backend

## What this project is about?

Codesnap Backend is a simple school-project for git mastering.

## How to run project?

1. clone the repo
1. install docker
1. open terminal with folder's directory
- run:

```bash
docker-compose up -d
```

- and:

```bash
npm run dev
```

## How to test?

- in terminal:

```bash
curl -i http://localhost:3000/health
```
